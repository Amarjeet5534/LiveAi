🚀 HOW TO RUN
═══════════════════════════════════════════════════════════════════

OPTION 1: LOCAL DEVELOPMENT (3 Terminals)
─────────────────────────────────────────

Terminal 1 (Ollama - if not already running):
  ollama serve

Terminal 2 (Backend):
  cd e:\LiveAI
  venv\Scripts\activate
  cd backend
  python fastapi_app.py

Terminal 3 (Frontend):
  cd e:\LiveAI
  cd frontend
  python -m http.server 3000

Then open: http://localhost:3000
Create account and start using!

OPTION 2: DOCKER COMPOSE (1 Command)
────────────────────────────────────

cd e:\LiveAI
docker-compose up -d

Then open: http://localhost:3000
Everything starts automatically!

OPTION 3: VOICE CLI (Optional)
──────────────────────────────

cd e:\LiveAI
venv\Scripts\activate
python main.py

Then speak commands:
  "open chrome"
  "search weather"
  "what is 2+2?"

🔒 SAFETY RULES
═══════════════════════════════════════════════════════════════════

BLOCKED COMMANDS:
❌ rm -rf, format, del /f /s
❌ shutdown, restart, taskkill
❌ pkill, cipher /w:, diskpart

BLOCKED PATTERNS:
❌ sudo, >>, ||, &&, ;

ALLOWED OPERATIONS:
✅ open [app]  - open chrome, open code
✅ search     - search weather, search pizza
✅ click       - click mouse
✅ scroll      - scroll down, scroll up
✅ type        - type text
✅ screenshot  - capture screen
✅ read file   - read C:\file.txt
✅ create file - create test.txt
