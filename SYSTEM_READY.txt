## 💾 Database Details

**Auto-Created Tables:**

```sql
-- Users table
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    username VARCHAR(50) UNIQUE,
    password_hash VARCHAR(255),
    created_at TIMESTAMP
)

-- Command history (shared by desktop + web)
CREATE TABLE command_history (
    id INTEGER PRIMARY KEY,
    user_id INTEGER,
    command VARCHAR(255),
    result TEXT,
    status VARCHAR(20),
    timestamp TIMESTAMP
)

-- Sessions (login tokens)
CREATE TABLE sessions (
    id INTEGER PRIMARY KEY,
    user_id INTEGER,
    token VARCHAR(255),
    created_at TIMESTAMP
)

-- Audit log (security)
CREATE TABLE audit_log (
    id INTEGER PRIMARY KEY,
    user_id INTEGER,
    action VARCHAR(255),
    timestamp TIMESTAMP
)
```

**View data:**
```bash
sqlite3 e:\LiveAI\data\app.db
SELECT * FROM users;
SELECT command, result FROM command_history LIMIT 10;
```

## 🔒 Safety & Security

✅ **Enabled by default:**
- Dangerous command blocking (rm -rf, shutdown, etc)
- User authentication (JWT tokens)
- Multi-user isolation (separate session data)
- Audit logging (all actions tracked)
- Password hashing (bcrypt)
- Session timeout (24 hours)

**Customize in backend/fastapi_app.py:**
```python
DANGEROUS_COMMANDS = [
    "rm -rf", "shutdown", "reboot", "sudo"
]

ALLOWED_COMMANDS = {
    "open chrome": "start chrome",
    "search for": "search_web"
}
```

## 🚨 Common Issues & Fixes

| Issue | Fix |
|-------|-----|
| "Module PyQt6 not found" | Run: `pip install PyQt6` |
| "Port 8000 in use" | Stop other apps or change port in .env |
| "Microphone not detected" | Run: `python verify_system.py` |
| "Web page shows blank" | Ensure Terminal 1 (Backend) is running |
| "Login failed" | Make sure you registered first |
| "Database is locked" | Restart all terminals |
| "Voice not transcribing" | Check Whisper installed: `pip install openai-whisper` |
